Binary Search

Enter the number you want to search for

Algorithm

binarySearch(x,arr,left,right)
     middle = (left + right)/2
     if(left > right)
         return -1
     else if(arr[middle] < x)
         return binarySearch(x,arr,middle+1,right)
     else if(arr[middle] > x)
         return binarySearch(x,arr,left,middle-1)
     else
         return middle

Analysis

The equation for the recursive algorithm is: `T(n)=T(n/2)+C`
at each recursive call the array size is divided in half and the work done outside the recursion call is constant which is calculating the middle index.
using master theorm where: `a=1,b=2`
Then: `f(n)=1`
`n^(log_b a)=n^(log_2 1)=1`
which means the complexity of the algorthim is
`Theta(logn)`
