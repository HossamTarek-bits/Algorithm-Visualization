Bubble Sort

Algorithm

bubbleSort(arr)
     for i from 0 to length(arr)-1
         for j from 0 to length(arr)-1-i
             if arr[j] > arr[j+1]
                 swap(arr[j],arr[j+1])

Analysis

for the first iteration the inner loop gets repeated n times.
for the second iteration the inner loop gets repeated n-1 times.
and so on until the final iteration the inner loop is done only once.
so the inner loop gets done `n,n-1,n-2,...,1`
solving the arthmitic series `sum_(i=1)^n i=(n(n+1))/2`
thus the time complexity for the algorithm is `Theta(n^2)`
